# ikos w3800

1. 设备支持通过赋予定位权限启动后台模式(无关紧要)
2. 支持通话自动/手动录音, 通话录音支持 icloud 同步, 支持播放和分享
3. 设备支持网线和 WiFi 使用远程模式
4. WiFi 不支持 5GHz, iPhone 个人热点难以连通, WiFi 的稳定性远低于网线
5. 蓝牙模式支持随时携带使用, 但是没有内置电池, 需要外部供电, 如充电宝
6. 未经电话支持通知推送提醒
7. 通话延迟取决于网络情况, 稳定的网络下几乎无延迟

## 代理设置

部分如`Stash`的代理软件采用 `fake-ip` 模式, 会导致相关的 udp 服务出现问题.

对应的表现是启动 `Stash` 后, 打开 App 无法远程登录设备. 关闭 `Stash` 后即可.

需要对 `w3800` 所使用的`simplus`通讯域名设置 fake-ip 白名单, 避免该问题

```yaml
dns:
    fake-ip-filter:
        # simplus
        - "+.unknowntec.com"
```
