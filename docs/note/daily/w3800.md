# ikos w3800

1. 设备貌似支持通过赋予定位权限启动后台模式
2. 支持通话自动/手动录音, 通话录音支持 icloud 同步, 支持播放
3. 设备支持网线和 WiFi 在线
4. 设备疑似支持蓝牙模式在离线状态下使用

## 代理设置

部分如`Stash`的代理软件采用 `fake-ip` 模式, 会导致相关的 udp 服务出现问题.

对应的表现是启动 `Stash` 后, 打开 App 无法远程登录设备. 关闭 `Stash` 后即可.

需要对 `w3800` 所使用的`simplus`通讯域名设置 fake-ip 白名单, 避免该问题

```yaml
dns:
    fake-ip-filter:
        # simplus
        - "+.unknowntec.com"
```
